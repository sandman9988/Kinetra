# Analysis Scripts

Research and analysis scripts for exploring market physics and strategy performance.

## Energy Analysis

- **`analyze_energy.py`** - Analyze market energy patterns
- **`analyze_energy_both.py`** - Comparative energy analysis
- **`analyze_energy_capture.py`** - Energy capture efficiency
- **`analyze_per_bar_energy.py`** - Per-bar energy breakdown

## Physics Analysis

- **`analyze_extended_physics.py`** - Extended physics indicators
- **`analyze_fat_candles.py`** - Fat candle analysis
- **`analyze_flow_entropy.py`** - Flow and entropy patterns
- **`analyze_reynolds_continuation.py`** - Reynolds continuation analysis
- **`superpot_physics.py`** - Superpot physics framework
- **`superpot_by_class.py`** - Superpot analysis by asset class
- **`superpot_complete.py`** - Complete superpot analysis
- **`superpot_explorer.py`** - Interactive superpot explorer

## Strategy Analysis

- **`analyze_asymmetric_rewards.py`** - Asymmetric reward analysis
- **`analyze_berserker_context.py`** - Berserker strategy context
- **`analyze_direction.py`** - Directional bias analysis
- **`analyze_directional_tension.py`** - Directional tension patterns
- **`analyze_reversal.py`** - Reversal pattern analysis
- **`analyze_trade_management.py`** - Trade management analysis
- **`analyze_triggers.py`** - Entry/exit trigger analysis

## Technical Analysis

- **`analyze_volatility_estimators.py`** - Volatility estimation methods
- **`pathfinder_deep_dive.py`** - Pathfinder strategy deep dive

## Utilities

- **`debug_csv.py`** - Debug CSV data issues
- **`quick_results.py`** - Quick results viewer

## Quick Start

```bash
# Analyze energy patterns
python scripts/analysis/analyze_energy_capture.py

# Explore superpot physics
python scripts/analysis/superpot_explorer.py

# Analyze trade management
python scripts/analysis/analyze_trade_management.py
```
